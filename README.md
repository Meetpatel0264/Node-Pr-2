# Student Management System

Student Management System is a simple CRUD-based web application developed using Node.js, Express.js, and EJS. The project is created to manage student records with a clean and easy-to-use interface.

This project includes basic operations like adding students, viewing student records, editing details, and deleting records.

---

## Technologies Used

* Node.js
* Express.js
* EJS
* Body Parser
* HTML5
* CSS3

---

## Features

* Add new student records
* View all students
* View individual student details
* Edit existing student information
* Delete student records
* Server-side rendering using EJS
* Clean and responsive user interface
* Simple folder structure

---

## Pages Included

* Home
* Add Student
* View Students
* View Student Details
* Edit Student

## Installation

### Clone Repository

```bash
git clone https://github.com/Meetpatel0264/Node-Pr-2
```

### Move to Project Folder

```bash
cd Node-Pr-2-main
```

### Install Dependencies

```bash
npm install
```

### Run Application

```bash
node index.js
```

or

```bash
npm start
```

### Open Browser

```bash
http://localhost:8000
```

## How It Works

### Home Page

The home page provides navigation to manage student records.

### Add Student

Users can add student details like name, email, course, and phone number.

### Student List

All student records are displayed with view, edit, and delete options.

### View Student

Shows complete details of a selected student.

### Edit Student

Allows users to update existing student information.
