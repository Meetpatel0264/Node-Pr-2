Student Management System

A simple and beginner-friendly Student Management System built using Node.js, Express.js, EJS, Body Parser, and CSS. This project allows users to manage student records with complete CRUD functionality through a clean web interface.

📌 Features
Add new student records
View all students
View individual student details
Edit existing student information
Delete student records
Responsive and clean user interface
Server-side rendering using EJS
🛠️ Technologies Used
Node.js
Express.js
EJS
Body Parser
HTML5
CSS3

🚀 Installation
1. Clone the Repository
git clone https://github.com/your-username/student-management-system.git
2. Navigate to Project Folder
cd student-management-system
3. Install Dependencies
npm install
4. Run the Application
node index.js
5. Open Browser
http://localhost:8000
